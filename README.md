# Overview

This project is a simple React app that displays a list of articles about  web development technologies with a search feature where users can search any word and related articles will returned to them with their matching words highlighted,

## Installation and Setup

First make sure you have node and npm installed on your machine, you can check using these commands: 

node --version

npm --version

you should get something like v22.19.0, if not please download node from here https://nodejs.org/en/download

Now follow these steps to download and run the project.

### `Clone the repo`

open your terminal and run this command:

git commit https://github.com/TheManana11/Text-Search.git

This will download a copy of the project files locally on your machine.

### `Navigate to project folder`

also in the terminal run these commands: 

cd Text-Search

cd text-search



### `Install Dependencies`

npm install

this will install all the requied packages to allow the project to run.


### `Run the Project`

npm start

This will run the project on http://localhost:3000/
